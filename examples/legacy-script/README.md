# Legacy Script Example

This example preserves the original usage model:

1. Define `window.PageConfig`
2. Include DemiurgeJS browser auto-bootstrap build
3. Let DemiurgeJS initialize on `DOMContentLoaded`

Use local built assets:
- `../../dist/demiurge.css`
- `../../dist/auto-bootstrap.global.js`

Run `npm run build` in the repo root before opening this example.

This example uses a local `page-config.js` and local `partials/` so it can run directly from the `examples/legacy-script/` folder.
